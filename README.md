# Predictive Analytics Using Historical Data

## Project Overview
This project predicts Walmart weekly sales using machine learning.

## Objective
Forecast future sales using historical data.

## Dataset
Walmart Store Sales Dataset

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Models
- Linear Regression
- Random Forest Regressor

## Results

Linear Regression
R² = 0.1555

Random Forest
R² = 0.9648

## Conclusion

Random Forest produced excellent prediction accuracy for weekly sales forecasting.
